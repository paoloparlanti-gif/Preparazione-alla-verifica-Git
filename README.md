# Esercitazione di preparazione alla verifica di laboratorio Git
Primo esercizio
Creare un repository Git in locale.
Effettuare almeno le seguenti modifiche (in una o più commit):
Aggiungere un nuovo file,
Modificare il contenuto del file.
Creare un repository su GitHub e collegarlo a quello locale.
Allineare il repository remoto su GitHub con le modifiche effettuate.
Consegnare su Classroom uno screenshot con i comandi eseguiti dalla bash e il repository locale.
Secondo esercizio
Creare una copia locale di questo repository .
Creare un nuovo branch di lavoro ed effettuare le seguenti modifiche al file .css (in più commit):
Modificare il colore di sfondo,
Modificare la dimensione del paragrafo,
Modificare l'allineamento del tag H1.
Riportare sul main solo la modifica i. e iii.
Allineare il repository remoto su GitHub con le modifiche effettuate.
Risolvere eventuali conflitti.
Consegnare su Classroom uno screenshot con i comandi eseguiti dalla bash.
